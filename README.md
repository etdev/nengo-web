[nengo](https://github.com/gazayas/nengo)のウェブ版 (http://nengo.io)

### 何のテクノロジーでできているか
* Rails 4.2, Ruby 2.2
* [Bootstrap 3](http://getbootstrap.com/)
* Slim

### このウェブサイトで何ができるか
* ~~和暦年号・西暦の変換~~
* ~~年号・西暦生まれから人の現在年齢チェック~~
* ~~年号・西暦の年の年齢情報チェック~~
